Getting Started

ROM Files

Bubilator88 loads ROMs from ~/Library/Application Support/Bubilator88/. This folder is created automatically on first launch.

Minimum Required File

FilePurposeNotes
N88.ROMN88-BASIC ROMThe app cannot start without this file.

Strongly Recommended Files

FilePurposeNotes
N80.ROMN-BASIC ROMIf it is missing, N-BASIC reads return 0xFF and boot can fail.
DISK.ROMSub-CPU disk ROMDisk access still works through a fallback path, but DISK.ROM gives more faithful behavior.
N88_0.ROM – N88_3.ROMN88 extension ROMsN88EXT0.ROM – N88EXT3.ROM are also accepted.

Optional Files

FilePurposeNotes
FONT.ROMFont ROMThe built-in ASCII font is used if it is missing.
KANJI1.ROMKanji ROM Level 1Used for Kanji display.
KANJI2.ROMKanji ROM Level 2Used for Kanji display.

Rhythm Sound WAV Files

To hear the YM2608 (OPNA) Rhythm part, place the following WAV files in the same folder. They must be signed 16-bit PCM. The app still runs without them, but the Rhythm part will be silent.

FileSound
2608_BD.WAVBass drum
2608_SD.WAVSnare drum
2608_TOP.WAVCymbal
2608_HH.WAVHi-hat
2608_TOM.WAVTom
2608_RIM.WAVRim shot
ROM files and Rhythm WAV files are copyrighted and are not included with Bubilator88. You must provide your own.

First Boot

After placing the ROM files, launch Bubilator88. ROMs are loaded on startup and again whenever you use Emulator > Reset.

To boot disk software, first insert a disk with Disk > Drive 1 > Mount... or Disk > Drive 1&2 > Mount..., then use Emulator > Reset (⌘E).

Boot Modes

Use Emulator > Boot Mode to select a mode. Changing it resets the emulator immediately.

ModeDescription
N88-BASIC V2Standard mode for SR and later. Most software uses this.
N88-BASIC V1HV1 high-speed mode. For software targeting models before the SR.
N88-BASIC V1SV1 standard mode. For software targeting models before the SR.
N-BASICPC-8001 compatibility mode. Uses N-BASIC ROM.

CPU Clock

Use Emulator > CPU Clock to switch between 8 MHz and 4 MHz. The setting is independent of Boot Mode, persists across launches, and is shown in the status bar.
