Using Disks

Supported Formats

Bubilator88 can open the following files:

  • D88 disk images
  • ZIP, LZH, CAB, and RAR archives (when they contain D88 disk images)

It also supports D88 files that contain multiple disk images.

Some RAR variants need unar, unrar, or 7z installed on your Mac. Direct 7z archive loading is not implemented.

Mounting Disks

MenuShortcut
Disk > Drive 1 > Mount...⌘1
Disk > Drive 2 > Mount...⌘2
Disk > Drive 1&2 > Mount...⌘3

Multiple Images and Archives

If the selected file contains multiple disk images, a picker sheet appears before mounting. After mounting, you can switch images from Disk > Drive 1 / Drive 2 or from the status-bar drive menus.

Drive 1&2 > Mount... assigns the first image to Drive 1 and the second image to Drive 2. If there is no second image, Drive 2 stays empty.

If an archive contains multiple disk files, you first choose which file to open.

Creating Blank Disks

Use Disk > Create Blank Disk... (⇧⌘N) to create a formatted blank D88 image. After saving, the new disk is mounted automatically in Drive 1.

  • 2D (320KB) — Single-sided double-density
  • 2DD (640KB) — Double-sided double-density (default)
  • 2HD (1.2MB) — Double-sided high-density

Ejecting and Recent Files

Use Disk > Drive 1 > Eject or Disk > Drive 2 > Eject to remove a disk.

Disk > Recent Files stores up to 10 recently opened files. Choosing one mounts it again in Drive 1&2 mode.
