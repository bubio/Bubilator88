Cassette Tape

Supported Formats

  • .cmt — raw byte-stream tape image
  • .t88 — tape image with data-carrier tags

Mounting a Tape

CommandShortcut
Tape > Open...⇧⌘T
Tape > Rewind—
Tape > Eject—

When a tape is mounted, the status bar shows a tape icon together with the current tape name (requires Settings > General > Show Tape in Status Bar).

Loading a Program

  1. Set Boot Mode to V1H (or V1S) so the N88-BASIC prompt appears.
  2. Choose a .cmt or .t88 file via Tape > Open....
  3. Type LOAD "CAS:" + Return in BASIC.
  4. When loading completes and Ok returns, type RUN + Return.

Recent Tapes

Tape > Recent Files keeps up to 10 of your most recently opened tapes.

Known Limitations

  • Writing to tape (SAVE "CAS:" and friends) is not supported. BASIC won't raise a Syntax error, but nothing is written to the tape.
  • Tapes with waveform-timing based copy protection may fail to load.
