Modern Macs do not have some PC-8801 keys directly. Change them in Settings > Keyboard > Special Key Mapping. The default mapping is:

Keyboard Layout

Choose Auto-detect, JIS, or US (ANSI) in Settings > Keyboard > Keyboard Layout. When Auto-detect is selected, the current detected layout is shown.

In JIS mode, several symbol keys are remapped to match JIS keycaps. For example, JIS @, [, and ] go to the matching PC-8801 keys.

Numpad Emulation

For keyboards without a physical numpad, two options are available in Settings > Keyboard:

  • Arrow Keys as Numpad — Maps arrow keys to numpad 2/4/6/8 for game character movement.
  • Number Row as Numpad — Maps the number row (0-9) to numpad 0-9 for games that only accept numpad input (e.g., adventure game menus).
