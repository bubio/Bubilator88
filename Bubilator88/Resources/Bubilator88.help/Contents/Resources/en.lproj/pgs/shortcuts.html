Keyboard Shortcuts

Emulator

ShortcutAction
⌘REmulator > Run / Stop
⌘EEmulator > Reset

Edit

ShortcutAction
⌘CEdit > Copy Screen (copy the screen as an image)
⇧⌘CEdit > Copy Text (copy on-screen text to macOS)
⌘VEdit > Paste Text (type the clipboard into the emulator)

Disk

ShortcutAction
⌘1Disk > Drive 1 > Mount...
⌘2Disk > Drive 2 > Mount...
⌘3Disk > Drive 1&2 > Mount...
⇧⌘NDisk > Create Blank Disk...

Tape

ShortcutAction
⇧⌘TTape > Open...

View

ShortcutAction
⌃⌘1View > Actual Size (x1)
⌃⌘2View > Double Size (x2)
⌃⌘4View > Quad Size (x4)
⌘TView > Translation Overlay

Control

ShortcutAction
⌘↑Control > Increase Volume
⌘↓Control > Decrease Volume
⇧⌘RControl > Start / Stop Audio Recording
⌘SControl > Quick Save
⌘LControl > Quick Load

Turbo Mode

ShortcutAction
⇧Tab (hold)Runs faster while held
Middle Click (hold)Runs faster while held

CPU Speed (Menu)

Use Control > CPU Speed to select a persistent speed multiplier: x1, x2, x4, x8, or x16.
