Display

Window Size

The base display resolution is 640 x 400 pixels. Use the View menu to change the window scale:

SizeShortcut
View > Actual Size (x1)⌃⌘1
View > Double Size (x2)⌃⌘2
View > Quad Size (x4)⌃⌘4

Use the standard macOS fullscreen control, usually the green window button.

Fullscreen

Choose the fullscreen scaling mode in Settings > Display > Scaling Mode:

  • Fit to Screen — Use as much screen space as possible while preserving aspect ratio.
  • Integer Scaling — Keep integer scaling for a pixel-perfect result, even with black borders.

In full screen, moving the mouse reveals a temporary bottom overlay with drive controls, volume, and the translation toggle.

Video Filter

Select a filter from View > Video Filter:

FilterDescription
NoneNearest-neighbor (sharp pixels)
LinearBilinear interpolation
BicubicSmooth bicubic interpolation
CRTCRT monitor simulation with phosphor and curvature
xBRZEdge-smoothing pixel art upscaler
EnhancedTwo-pass: de-dithering + xBRZ
AI Upscale (Fast)Lightweight SRVGGNet (32ch × 12 layers, ~115K params). Prioritizes real-time speed
AI Upscale (Balanced)Standard SRVGGNet (64ch × 16 layers, ~600K params). Balanced quality and speed
AI Upscale (Quality)Real-ESRGAN. Highest quality, best for static screens

Scanlines

Toggle a scanline overlay with View > Scanlines. It is available only with None, Linear, and Bicubic.

Translation Overlay

Toggle it with ⌘T or View > Translation Overlay. The source language is Japanese.

When enabled, the app runs OCR immediately on the current frame, then re-runs OCR roughly every 3 seconds when the screen content changes. Set the target language in Settings > Display > Target Language.

The required translation language data must be downloaded in macOS System Settings.

Screenshots

Use Control > Save Screenshot... to save the current frame. The file format is chosen in Settings > General > Format and can be PNG, JPEG, or HEIC.
