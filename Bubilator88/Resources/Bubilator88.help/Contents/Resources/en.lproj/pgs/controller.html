Game Controller

Setup

Connect a controller supported by GameController.framework and enable Settings > Controller > Enable Game Controller. With one controller connected, its name is shown directly. With multiple controllers, use Active Controller to choose one.

Button Mapping

When an active controller is present, a Button Mapping section appears. Click the button you want to change, then press the keyboard key you want to assign.

Button labels and icons adapt automatically to the connected controller. Mappings are stored for each controller.

Use the ✕ icon to clear a single assignment, or Reset to Defaults to restore the default map.

Haptic Feedback

Enable Settings > Controller > Enable Haptic Feedback to vibrate the controller when the app detects SSG noise-based effect sounds. A haptics-capable controller is required.
