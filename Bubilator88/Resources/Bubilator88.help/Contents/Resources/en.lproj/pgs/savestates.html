Save States

Quick Save / Load

ActionShortcut
Control > Quick Save⌘S
Control > Quick Load⌘L

Quick Save stores one snapshot. When it exists, the Control menu shows the saved time and mounted disk names. Quick Load is enabled only when a quick save exists.

Save State / Load State

Open the slot manager with Control > Save State... or Control > Load State.... There are 10 slots, each with a thumbnail, timestamp, and mounted disk names.

In Load State mode, empty slots are disabled.

What Is Saved

Save states include:

  • CPU registers and flags
  • All RAM and VRAM
  • Sound chip registers (YM2608)
  • FDC and disk controller state
  • I/O port state and interrupt controller
  • Boot Mode and CPU Clock
  • The mounted disk images themselves and the contents they had when you saved
Save states include the mounted disk images, so you do not need to remount the same disk before loading.

State files are stored in ~/Library/Application Support/Bubilator88/SaveStates/.
