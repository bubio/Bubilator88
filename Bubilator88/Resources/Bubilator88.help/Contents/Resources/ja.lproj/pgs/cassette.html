カセットテープ

対応フォーマット

  • .cmt — バイト列そのままのテープイメージ
  • .t88 — データキャリアタグ付きテープイメージ

テープのマウント

操作ショートカット
テープ > 開く...⇧⌘T
テープ > 巻き戻し—
テープ > 取り出し—

テープをマウントすると、ステータスバーにテープアイコンと現在のテープ名が表示されます（設定 > 一般 > ステータスバーにテープを表示 が有効なとき）。

読み込み手順

  1. ブートモードを V1H（もしくは V1S）にして N88-BASIC プロンプトを出します。
  2. テープ > 開く... から .cmt または .t88 を選択。
  3. BASIC に LOAD "CAS:" + Return を入力。
  4. 読み込みが終わり Ok に戻ったら RUN + Return。

最近使ったテープ

テープ > 最近使ったファイル から、最近開いたテープを最大 10 件までたどれます。

既知の制限

  • テープへの書き込み（SAVE "CAS:" など）には対応していません。BASIC は Syntax error にはなりませんが、テープには何も記録されません。
  • 波形タイミングに依存するプロテクトが掛かったテープは読み込めないことがあります。
