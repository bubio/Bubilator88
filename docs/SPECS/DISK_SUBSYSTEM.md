# PC-8801 ディスクサブシステム

PC-8801 のディスクアクセスは、メインCPUから直接 FDC を操作するのではなく、
サブCPU (Z80) + PIO (8255) を介したメッセージパッシング方式で行う。

---

## 1. 構成

```
メインCPU (Z80)                    サブCPU (Z80)
     │                                  │
     └── PIO 8255 (Port 0xFC-0xFF) ──┘
              クロスワイヤ接続
                                    サブCPU
                                      │
                                FDC uPD765A
                                      │
                                 フロッピードライブ
```

メインCPU は PIO のハンドシェイクプロトコルを通じてサブCPUにコマンドを送信し、サブCPUが FDC を操作する。

---

## 2. PIO I/Oポート (メインCPU側)

| Port | R/W | 機能 |
|------|-----|------|
| 0xFC | R/W | ポートA — サブシステムからのデータ受信 |
| 0xFD | R/W | ポートB — サブシステムへのデータ送信 |
| 0xFE | R/W | ポートC — ハンドシェイク信号 |
| 0xFF | W | コントロールレジスタ |

### ポートC ビット配置

| Bit | R/W | 信号 | 説明 |
|-----|-----|------|------|
| 7 | W | ATN | コマンド送信開始 / 中断時に 1 |
| 6 | W | DAC | データ引取り完了時に 1 |
| 5 | W | RFD | データ要求時に 1 |
| 4 | W | DAV | ポートB にデータ出力後に 1 |
| 2 | R | DAC | サブがデータ引取り完了で 1 |
| 1 | R | RFD | サブがデータ要求で 1 |
| 0 | R | DAV | ポートA にデータ出力済みで 1 |

---

## 3. 通信プロトコル

### コマンド送信

1. メイン: ATN = 1
2. サブ: ATN=1 を検出 → RFD = 1
3. メイン: RFD=1 を検出 → ATN = 0
4. メイン: データをポートBに出力
5. メイン: DAV = 1
6. サブ: DAV=1 を検出 → RFD = 0, データ受取
7. サブ: DAC = 1
8. メイン: DAC=1 を検出 → DAV = 0
9. サブ: DAV=0 を検出 → DAC = 0

### 通常ハンドシェイク

1. サブ: RFD = 1
2. メイン: RFD=1 を検出 → データをバスに出力
3. メイン: DAV = 1
4. サブ: DAV=1 を検出 → RFD = 0, データ受取
5. サブ: DAC = 1
6. メイン: DAC=1 を検出 → DAV = 0
7. サブ: DAV=0 を検出 → DAC = 0

### 高速ハンドシェイク

通常ハンドシェイクの変形。DAC=1 を検出した時点で次のデータをバスに乗せ、DAV=0 で受信側が2バイト目を受取る。

---

## 4. ROM内通信ルーチン

| ラベル | N88アドレス | N アドレス | 機能 |
|--------|-----------|----------|------|
| PUTCOM | 37CCh | 017Ch | Aレジスタをコマンドとしてサブに送信 |
| PUTDAT | 37D2h | 0183h | Aレジスタをデータとしてサブに送信 |
| GETDAT | 3847h | 01E9h | サブからデータをAレジスタに受信 |

※ N88モードではワークエリア (EF15h) が非0のとき、Lレジスタの内容を2バイト目として高速ハンドシェイクを行う。

---

## 5. サブシステムコマンド一覧

コマンド番号はコマンド送信プロトコルで、パラメータは通常ハンドシェイクで送る。

| Cmd | 名前 | 形式 | 説明 |
|-----|------|------|------|
| 00h | イニシャライズ | `<00h>` | モータON, FDCリセット, ヘッド初期化 |
| 01h | ライトデータ | `<01h> <セクタ数> <DRV> <TRK> <SEC> <データ...>` | ディスクに書込み |
| 02h | リードデータ | `<02h> <セクタ数> <DRV> <TRK> <SEC>` | ディスクから読出し (バッファに格納) |
| 03h | センドデータ | `<03h>` → `<データ...>` | リードしたデータをホストに転送 |
| 04h | コピー | `<04h> <数> <DRV1> <TRK1> <SEC1> <DRV2> <TRK2> <SEC2>` | セクタコピー |
| 05h | フォーマット | `<05h> <DRV>` | ディスクフォーマット |
| 06h | センドリザルト | `<06h>` → `<ステータス>` | 直前のコマンド結果 |
| 07h | センドドライブ | `<07h>` → `<ステータス>` | ドライブ接続状態 |
| 08h | テストメモリ | `<08h>` → `<ステータス>` | サブRAMテスト (80h=OK, 81h=NG) |
| 09h | センドメモリ | `<09h> <開始H> <開始L> <終了H> <終了L>` → `<データ>` | サブRAMをホストへ |
| 0Ah | アウトマージン | `<0Ah> <データ>` | ポートF7hに出力 |
| 0Bh | センドメモリ2 | `<0Bh> <開始H> <開始L> <バイト数H> <バイト数L>` → `<データ>` | サブRAMをホストへ |
| 0Ch | レシーブメモリ | `<0Ch> <開始H> <開始L> <バイト数H> <バイト数L>` | ホストからサブRAMへ |
| 0Dh | エグゼキュート | `<0Dh> <開始H> <開始L>` | サブで指定アドレスから実行 |
| 0Eh | ロードデータ | `<0Eh> <数> <DRV> <TRK> <SEC> <開始H> <開始L>` | ディスク→サブRAM |
| 0Fh | セーブデータ | `<0Fh> <数> <DRV> <TRK> <SEC> <開始H> <開始L>` | サブRAM→ディスク |
| 10h | ロード&ゴー | `<10h>` | Drive1 Track0 Sec1 → 5000h に読込み実行 (IPL) |
| 11h | 高速ライト | 01h と同じ (高速ハンドシェイク) | |
| 12h | 高速センド | 03h と同じ (高速ハンドシェイク) | |
| 13h | エラーインフォ | `<13h>` → `<CMD> <ST0> <ST1> <ST2> <C> <H> <R> <N>` | FDCステータス |

### リザルトステータス (Cmd 06h)

| Bit | 説明 |
|-----|------|
| 7 | I/O動作終了 |
| 6 | リードバッファにデータあり |
| 0 | エラー |

### ドライブステータス (Cmd 07h)

| Bit | 説明 |
|-----|------|
| 7-4 | ドライブ接続状態 (各ビットが各ドライブ) |

---

## 6. サブCPU I/Oポート

| Port | R/W | 機能 |
|------|-----|------|
| F4h | W | ドライブ/サイド選択 |
| F7h | W | プリンタポート出力 |
| F8h | R | FDC ターミナルカウント (TC信号発行) |
| F8h | W | モータ制御 (bit0=Drive0, bit1=Drive1) |
| FAh | R | FDC メインステータス (uPD765A MSR) |
| FBh | R/W | FDC データレジスタ (uPD765A) |
| FC-FFh | R/W | PIO 8255 (サブ側) |
